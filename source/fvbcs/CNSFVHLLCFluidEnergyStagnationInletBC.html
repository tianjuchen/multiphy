CNSFVHLLCFluidEnergyStagnationInletBC

Adds the boundary fluid energy flux for HLLC when provided stagnation temperature and pressure

Overview

This object implements the fluid energy portion of a stagnation boundary condition based on stagnation pressure and stagnation temperature. Please see CNSFVHLLCStagnationInletBC for the equations and theory behind computing boundary conditions based on stagnation temperature and pressure.

    Controllable:No

    Description:The list of boundary IDs from the mesh where this boundary condition applies

    Description:Fluid properties userobject

  • stagnation_pressureSpecified inlet stagnation pressure.

    C++ Type:PostprocessorName

    Description:Specified inlet stagnation pressure.

  • stagnation_temperatureSpecified inlet stagnation temperature.

    C++ Type:PostprocessorName

    Controllable:No

    C++ Type:NonlinearVariableName

    Controllable:No

    Description:The name of the variable that this boundary condition applies to
