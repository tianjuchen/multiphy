FunctionIntegralRayKernel

Integrates a function along a Ray.

The integration performed for the function is

which is achieved by overriding computeQpIntegral() as:

Real
FunctionIntegralRayKernel::computeQpIntegral()
{
  return _function.value(_t, _q_point[_qp]);
}
(moose/modules/ray_tracing/src/raykernels/FunctionIntegralRayKernel.C)

The resulting integrated value can be obtained using a RayIntegralValue postprocessor.
