GapFluxModelRadiative

Gap flux demonstration model for radiative heat conductance

Description

commentnote:Not Recommended for Use

This particular class, GapFluxModelRadiative represents an early implementation of the modular gap flux constraint system and has limited functionality. The more robust radiative heat flux class, GapFluxModelRadiation is recommended for use instead.

GapFluxModelRadiative computes a radiative heat flux across a gap. It is used by ModularGapConductanceConstraint.
