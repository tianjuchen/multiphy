SoundspeedMat

Computes the speed of sound

Overview

This object simply computes the sound_speed material property given a SinglePhaseFluidProperties derived object specified by the fp parameter. This object relies on pressure and fluid temperature being computed as material properties by some other material object.

    Description:fluid userobject
