INSFVInletVelocityBC

This object simply wraps FVFunctionDirichletBC, so a required parameter is function describing the velocity along an inlet boundary. The variable parameter should correspond to a velocity component variables. If applying INSFVInletVelocity for any velocity component on a given boundary, then an INSFVInletVelocity should be specified for every velocity component on that boundary. A FVBC for pressure should not be applied on the same boundary. Instead a value for the pressure at the inlet will be extrapolated from the interior.
