SaturationPressureMaterial

This material computes saturation pressure as a material property from a temperature material property and the p_sat(T) interface of a TwoPhaseFluidProperties object.

Input Parameters

  • TTemperature material property

    C++ Type:MaterialPropertyName

    Controllable:No

    Description:Temperature material property

  • fp_2phaseTwo-phase fluid properties user object name

    C++ Type:UserObjectName

    Controllable:No

    Description:Two-phase fluid properties user object name

  • p_satName to give saturation pressure material property

    C++ Type:MaterialPropertyName

    Controllable:No

    Description:Name to give saturation pressure material property
