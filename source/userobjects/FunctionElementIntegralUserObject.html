FunctionElementIntegralUserObject

Computes a volume integral of a function.

The FunctionElementIntegralUserObject is mostly used to apply an integration operation to a function when creating other user objects by inheriting from both this class, for the integration operation, and another class that defines how to interact with the mesh to create custom spatial integration user objects, such as the FunctionLayeredIntegral.

Used on its own, this user object performs a similar role as the FunctionElementIntegral, except that it does not set up the default output to console and CSV. Using the postprocessor is the preferred way of performing this spatial function integration.
