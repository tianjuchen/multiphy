FirstPiolaKirchhoffStress

This class computes the first Piola-Kirchhoff stress associated with given energy densities.

Overview

The first Piola-Kirchhoff stress is defined as where is the deformation gradient, is the total internal energy density, and is the total dissipation density.

Example Input File Syntax

[pk1_stress]
  type = FirstPiolaKirchhoffStress
  first_piola_kirchhoff_stress = P
  deformation_gradient_rate = dot(F)
