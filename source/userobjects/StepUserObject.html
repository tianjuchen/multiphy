StepUserObject

Description

StepUserObject is a general user object that performs basic computations to provide a loading step given a simulation time step value or a simulation time step value given a loading step. This user object is used to interface with AbaqusUMATStress and AbaqusUExternalDB to provide step information to user routines. StepUserObject can also interface with StepPeriod to enable/disable boundary conditions and constraints according the user-defined loading steps.

Example Input File Syntax

[step_uo]
  type = StepUserObject
  step_start_times = '0 5'
[]
(moose/modules/tensor_mechanics/test/tests/umat/steps/elastic_temperature_steps_uo.i)
